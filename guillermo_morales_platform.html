Paso 2 — Redirección segura a Transbank
// 1. INICIAR transacción (POST /api/webpay/init) const { token, url } = await WebpayPlus.Transaction.create({ buyOrder: "GM-2024-0190", sessionId: "sess_abc123", amount: 84500, // CLP — sin decimales returnUrl: "https://gmotors.cl/webpay/return" }); // → token: "01ab45...", url: "https://webpay3g.transbank.cl/..." // 2. REDIRIGIR al portal de Transbank (formulario POST) `<form action="${url}" method="POST"> <input name="token_ws" value="${token}"> <script>document.forms[0].submit()</script> </form>` // 3. RECIBIR token de retorno (GET /webpay/return?token_ws=...) const tokenWs = req.query.token_ws; // o req.form_params // 4. CONFIRMAR transacción (obligatorio dentro de 30s) const response = await WebpayPlus.Transaction.commit(tokenWs); // response.responseCode === 0 → APROBADA // response.responseCode !== 0 → RECHAZADA / ANULADA // 5. ACTUALIZAR BD y notificar usuario if (response.responseCode === 0) { await db.orders.update({ status: "PAGADA", authCode: response.authorizationCode }); await sendConfirmationEmail(response); } else { await db.orders.update({ status: "RECHAZADA" }); }
Pago aprobado
responseCode: 0
authorizationCode: "ABC123"
→ Orden marcada PAGADA
→ Email de confirmación enviado
Pago rechazado / error
responseCode: -1, -2, etc.
→ Stock se revierte (+1)
→ Orden marcada RECHAZADA
→ Notificar al usuario
